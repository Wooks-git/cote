```python
i = 0
while True:
    L, P, V = map(int, input().split())
    answer = 0
    i += 1

    if L == 0 and P == 0 and V == 0:
        break
    
    
    answer += (V//P)*L + min(V%P, L)
    # answer += V%P # 나머지가 남아있는 휴가보다 클 수 있으므로 해당 연산은 오류

    print(f"Case {i}: {answer}")
```
